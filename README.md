# Gesture-Controlled Robot

Цей проєкт демонструє робота, керованого жестами рук за допомогою бездротового передавача та приймача на базі D1 Mini та ESP32 Wroom32.

## Проєкт
Робот отримує сигнали від сенсора руху/жестів на передавачі (D1 Mini) і відповідно керує рухом робота через приймач (ESP32 Wroom32).  
Можна виконувати базові команди: вперед, назад, поворот, зупинка.

## Компоненти
- **Передавач:** D1 Mini з сенсором жестів (`GestureRobotTransmitter/`)
- **Приймач:** ESP32 Wroom32, двигуни/сервоприводи (`GestureRobotReceiver/`)
- **Корпус:** STL-файли для друку корпусів робота та передавача (`STL/`)
- **Схеми:** KiCad-проекти для передавача та приймача (`KiCad/`)

## Інструкція запуску
1. Встановити **VS Code** з **PlatformIO**.
2. Відкрити папку `GestureRobotTransmitter/` і завантажити код на **D1 Mini**.
3. Відкрити папку `GestureRobotReceiver/` і завантажити код на **ESP32 Wroom32**.
4. Підключити живлення до обох плат.
5. Використовувати сенсор жестів для керування роботом:
   - Рух руки вперед → робот їде вперед  
   - Рух руки назад → робот їде назад  
   - Рух руки вліво/вправо → поворот робота  
   - Зупинка руки → зупинка робота

## Файли репозиторію

- `GestureRobotReceiver/` – код приймача на ESP32

- `KiCad/` – проєкти плат
